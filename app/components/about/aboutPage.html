    About the site

    This site was created as part of assignment to develop Online Shop (e-commerce).
    This site based on clinet-server architecture where the client implemented in Angular and the server implemented in NodeJS using Microsoft Azure Cloud SQL Database.
    The hard part of this assignments was to reduce the number of calls to the server, to allow the costumer smooth stay in the site without waiting for uploading each page. Also it was the first time we develop client side using Angular that provide a huge possibilities that we are not familiar with them , but we discovered them :) .
